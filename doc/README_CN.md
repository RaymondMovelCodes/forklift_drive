# forklift_drive 使用说明

## 功能描述

叉车驱动。

## 开始 

**1.编译package**

将forklift_drive放在工作空间，例如catkin_ws,然后编译：

```
cd ~/catkin_ws
catkin_make
```

**2.配置forklift_drive.launch文件参数**

```
<param name="   " value="" />
<param name="   " value="10" />
```

* “   ” ： 。

* "   " ： 。

**3.运行程序**

**【1】运行感知程序：**

`roslaunch rs_launch forklift_drive.launch`
